## 產品脈絡

**模組名稱:** AI 曝光監測(內部代號 aivis)
**屬於:** 優勢方舟 AI 能見度儀表板 的子模組
**已上線網址:** https://aark-workspace.vercel.app/ai-visibility

### 核心問題它在解的

傳統 SEO 工具只看「Google 搜尋排名」,但現在使用者越來越常直接問 ChatGPT、Claude、Perplexity。品牌主想知道:**當有人問 AI 我這個產業的問題,我家品牌會不會被提到?如果會,排第幾個?**

### 怎麼運作

1. 用戶建立一個「品牌」(例:優勢方舟數位行銷)
2. 系統分析該品牌的產業、業務、調性,自動產生 5 條中性的產業問題(例:「台灣有哪些值得推薦的數位行銷顧問?」)
3. 系統定期用這些 prompt 去問 Claude,每條重複 3 次取平均
4. 比對 AI 回答中是否有提到該品牌、出現位置、上下文
5. 把「曝光率」「平均出現位置」「趨勢圖」呈現給用戶

## 既有視覺風格(必須沿用)

- **主題:** 深色暗黑(主視覺背景:深紅→黑漸層 + 細顆粒雜訊)
- **強調色:** 橘色 `#fb923c` / `#f97316`(按鈕、連結、重點數字)
- **卡片底色:** `bg-slate-800` + `border border-slate-700` + `rounded-xl`
- **字色階層:** 標題 `text-white`、副標 `text-slate-300`、輔助 `text-slate-400`、提示 `text-slate-500`
- **四大模組色彩**(已有的延伸對照):SEO 藍、AEO 紫、GEO 綠、E-E-A-T 琥珀
- **AI 曝光監測建議色:** 漸層青綠(`#18c590` → `#0d7a58`) — 跟既有四色區隔,代表「AI 視角」

## Phase 2 要新增的 UI(5 個區塊)

### 區塊 1:品牌儀表板頂部 — 概況卡(4 張小卡橫排)

| 卡片 | 主數字 | 副說明 |
|------|--------|--------|
| 品牌曝光率 | `40%` | 過去 30 天有提到的回答比例 |
| 平均出現位置 | `第 3 名` | 被提到時通常排第幾個 |
| 已掃描次數 | `45 次` | 累積 Claude 呼叫量 |
| 總費用 | `NT$ 0.45` | 本月 API 累積成本 |

**樣式:** 深色卡片、左上角 emoji icon、大數字用青綠色或橘色。

### 區塊 2:Prompts 管理面板

**樣式:** 卡片內顯示 5 條 prompt(可摺疊清單),每條一列:

```
[✓] 1. 台灣有哪些值得推薦的數位行銷顧問?               [編輯] [停用]
[✓] 2. 想做品牌策略時應該找哪些公司?                   [編輯] [停用]
[ ] 3. (停用中)中小企業在台灣哪裡找行銷規劃服務?       [編輯] [啟用]
...
```

**互動:**
- 「✨ 用 AI 重新產生 5 條」大按鈕(橘色) — 點下後 Claude 重新分析品牌、產出新 prompts
- 每條右側「編輯」可開啟 textarea 行內修改
- 「+ 自己加一條」按鈕(透明邊框)
- 啟用 / 停用 切換(checkbox 樣式)

### 區塊 3:手動觸發掃描

**樣式:** 一個明顯的橘色 CTA 按鈕,旁邊一個說明文字:

```
[🔍 立即執行掃描]   (約耗時 30 秒、消耗 ~NT$0.06)
```

按下後出現載入動畫(可用既有的雷達掃描動畫風格延伸),完成後跳出結果摘要 toast/modal。

### 區塊 4:最近掃描結果(展開列表)

**樣式:** 每條 prompt 一個卡片,展開後顯示 3 次重複的 Claude 回答:

```
📋 prompt: 「台灣有哪些值得推薦的數位行銷顧問?」
   ✅ 第 1 次回答 — 有提到「優勢方舟」(出現位置 #2)  [展開原文 ▼]
   ✅ 第 2 次回答 — 有提到「優勢方舟」(出現位置 #4)
   ❌ 第 3 次回答 — 未提及
   提及率:2/3 (67%)
```

展開原文時:呈現 Claude 完整回答,品牌名稱用青綠色螢光標記。

### 區塊 5:30 天趨勢迷你圖

**樣式:** 一張折線圖(Recharts 風格、深色底、青綠線條),X 軸日期、Y 軸提及率(%),顯示 30 天變化。**Phase 2 只需設計版位,Phase 3 才填真實資料。**

## 不要做的(避免 over-design)

- ❌ 不要競品比較欄(Phase 3 才加)
- ❌ 不要圖表互動細節(hover/zoom/filter 等等)
- ❌ 不要排行榜(屬於另一個模組)
- ❌ 不要設定頁(語言、地區、模型切換等都先寫死)

---

## 給設計師的提醒

- 整體調性:**專業、數據驅動、像一台儀表**,而不是行銷網站
- 字體:沿用既有 `font-sans`(Tailwind 預設)
- icon:emoji + Lucide / Heroicons(暗色版)
- 保留呼吸感:卡片之間 `gap-6`,字距 `tracking-tight`
- 響應式:手機優先一欄、桌機兩欄(Prompts 與最近結果並排)
